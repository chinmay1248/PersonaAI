================================================================================
                         IF SOMETHING FAILS
================================================================================

"Command not recognized"
  → Node.js not installed
  → Install from: https://nodejs.org/

"npm: command not found"
  → npm not in PATH
  → Restart Command Prompt or reinstall Node.js

"npm install" fails
  → Run: npm cache clean --force
  → Then: npm install

"eas: command not found"
  → Run: npm install -g eas-cli
  → Wait for installation
  → Try build again

"EAS build fails"
  → Check internet connection
  → Run: npm cache clean --force
  → Try build again

"Build takes more than 15 minutes"
  → This is normal! Keep waiting
  → Don't close the window
  → Build is happening in the cloud

"Can't find download link in terminal"
  → Look for blue text containing "https://eas-builds..."
  → Scroll up if needed
  → Copy the full link

"APK won't install on phone"
  → Uninstall previous version first
  → Enable "Unknown Sources" in Settings → Security
  → Try install again

"App crashes when opening"
  → Uninstall and reinstall APK
  → Force stop app: Settings → Apps → PersonaAI → Force Stop
  → Clear cache: Settings → Apps → PersonaAI → Clear Cache

"Can't login with demo credentials"
  → Check phone has internet
  → Check Railway backend is running
  → Try again in 30 seconds
