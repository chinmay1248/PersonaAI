# Docs Map

This folder is the main documentation entry point for the repo.

## Recommended Reading Order

1. [guides/QUICK_START.md](guides/QUICK_START.md) for local setup.
2. [guides/DEPLOYMENT.md](guides/DEPLOYMENT.md) for backend and production deployment.
3. [guides/PHONE_SETUP_GUIDE.md](guides/PHONE_SETUP_GUIDE.md) if you want the Android and WhatsApp flow.
