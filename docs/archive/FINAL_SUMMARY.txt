IMPORTANT NOTES
================================================================================

1. BACKEND REQUIRED
   PersonaAI needs a backend server to work.
   
   Option A: Local (Development)
     - Start in backend folder: python -m uvicorn app.main:app
     - Update .env: EXPO_PUBLIC_API_URL=http://localhost:8000/v1
     - Pro: Fast, free
     - Con: Must keep running
   
   Option B: Railway (Production)
     - Deploy to Railway first
     - Get URL: https://your-domain.up.railway.app
     - Update .env: EXPO_PUBLIC_API_URL=https://your-domain.up.railway.app/v1
     - Pro: Always available
     - Con: ~$5-10/month

2. FILE TRANSFER METHODS
   Choose one way to get APK to phone:
   - USB Cable (fastest, 1 min)
   - Email (easy, 2-5 min)
   - Google Drive (convenient, 3-5 min)
   - QR Code (instant, scan from EAS page)

3. FIRST LAUNCH
   App will:
   - Show login screen
   - Ask to create account or demo login
   - Show 4-step onboarding
   - Request permissions
   - Be ready to use!

4. PERMISSIONS NEEDED
   Phone will ask for:
   - Network access (required)
   - Storage access (optional, for uploads)
   - Accessibility (optional, for WhatsApp training)

5. AUTO-TRAINING (OPTIONAL)
   To enable WhatsApp message learning:
   - Go to Settings → Auto-Training → Enable
   - Android Settings → Accessibility → PersonaAI → Enable
   - Grant "Read Screen Content" permission
   - Wait 15 minutes for first training
